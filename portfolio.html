These are the portfolio websites I have completed or am working on. Feel free to look around my websites

BookEat

BookEat is an efficient and easy to use reservation system which is based on a web application. The purpose of this web application is to make communication between customers and restaurants easier. With BookEat people can make a reservation in their favourite restaurant within 1 minute. Also, restaurant owners can easily manage reservations in one single page.

  • Frontend: React
    • Frontend Github
  • Backend: NodeJS
    • Backend Github
  • Database: MongoDB
  • Platform: Heroku
  • Tools: Git, Github and Visual Studio Code

Music Management Website
